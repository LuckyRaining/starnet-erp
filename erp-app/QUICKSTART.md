## ⚠️ 使用前必须完成的步骤

### 1. 准备TabBar图标 (必需!)

在 `static/tabbar/` 目录下创建以下10个PNG图标文件(尺寸81x81像素):

**未选中状态:**
- `home.png` - 首页图标
- `product.png` - 商品图标  
- `checkout.png` - 收银图标
- `statistics.png` - 统计图标
- `mine.png` - 我的图标

**选中状态(建议使用主题色 #2979ff):**
- `home-active.png`
- `product-active.png`
- `checkout-active.png`
- `statistics-active.png`
- `mine-active.png`

> 💡 **提示**: 可以从iconfont.cn下载或使用设计工具制作。当前目录只有`.placeholder`占位文件。

### 2. 配置后端API地址

编辑 `utils/config.js`:
```javascript
const CONFIG = {
    BASE_URL: 'http://localhost:9090',  // 修改为实际的后端地址
    // ...
}
```

### 3. 确保后端服务运行

启动 erp-api 后端服务,确保以下接口可访问:
- `POST /user/login` - 登录
- `POST /product/page` - 商品列表
- `POST /category/list` - 分类列表
- `POST /analysis/sale/detailList` - 销售明细
- `POST /checkout/createCode` - 创建出库单号
- `POST /checkout/save` - 保存出库单

## 🎯 核心功能演示流程

### 场景1: 完整收银流程

1. **登录系统**
   - 点击"我的" -> "点击登录"
   - 输入: admin / 123456
   - 登录成功返回首页

2. **浏览商品**
   - 切换到"商品"Tab
   - 左侧选择分类
   - 浏览右侧商品列表
   - 点击商品添加到购物车

3. **查看购物车**
   - 切换到"收银"Tab
   - 查看已添加的商品
   - 调整数量(+/-按钮)
   - 或删除不需要的商品

4. **挂单(可选)**
   - 点击"挂单"按钮
   - 当前购物车被保存
   - 可在"我的"->"挂单记录"中查看

5. **结算**
   - 确认购物车商品无误
   - 点击"结算"按钮
   - 系统自动生成出库单
   - 显示订单号和总金额
   - 购物车自动清空

### 场景2: 数据统计查询

1. 切换到"统计"Tab
2. 选择日期范围(默认最近7天)
3. 点击"查询"按钮
4. 查看统计数据:
   - 总销售额
   - 订单数量
   - 平均客单价
   - 商品销售排行柱状图
   - 销售趋势折线图

### 场景3: 挂单取单

1. **挂单**:
   - 在收银页添加商品到购物车
   - 点击"挂单"按钮
   - 购物车清空,挂单已保存

2. **取单**:
   - 切换到"我的"Tab
   - 点击"挂单记录"
   - 找到对应的挂单
   - 点击"取单"按钮
   - 自动跳转到收银页,商品已恢复

## 🐛 常见问题解决

### Q1: TabBar不显示图标
**A**: 检查 `static/tabbar/` 目录下是否有10个PNG图标文件,文件名必须完全匹配。

### Q2: 请求后端API失败
**A**: 
1. 确认后端服务已启动
2. 检查 `utils/config.js` 中的BASE_URL是否正确
3. 查看浏览器控制台网络请求详情
4. 确认跨域配置(CORS)已启用

### Q3: 登录成功后仍然提示未登录
**A**: 
1. 检查浏览器Storage中是否有`erp_token`
2. 确认 `utils/user.js` 中的setToken方法被正确调用
3. 刷新页面重试

### Q4: Canvas图表不显示
**A**: 
1. 确认有销售数据返回
2. 检查Canvas的canvas-id是否正确
3. 查看控制台是否有绘图错误

### Q5: 扫码功能无法使用
**A**: 
1. 确认在真机或模拟器中测试
2. 检查是否授予摄像头权限
3. H5环境不支持扫码,需使用微信小程序环境
